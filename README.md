## About

This portfolio website is designed to highlight my work, skills, and experience. It provides a simple and interactive interface for visitors to learn about me and get in touch.

## Features

- Responsive design for mobile, tablet, and desktop
- Showcase of my projects with live links and descriptions
- About me section with skills and experience
- Contact form to get in touch directly
- Smooth scrolling and modern UI effects

## Technologies Used

- **ReactJS** - Frontend library
- **CSS / SCSS** - Styling
- **React Router** - Navigation
- **EmailJS / Formspree** (optional) - Contact form functionality
